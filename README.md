# Gerenciamento de Dados da COVID-19

Projeto desenvolvido em linguagem C com o objetivo de analisar a relação entre os índices de isolamento social e o surgimento de novos casos de COVID-19 durante o mês de junho de 2020. O sistema realiza o gerenciamento de dados epidemiológicos a partir de arquivos `.txt`, permitindo consultas, comparações e análises temporais dos registros armazenados.

Para o desenvolvimento do projeto, foi utilizada uma estrutura de dados do tipo lista simplesmente encadeada com nó cabeçalho (descritor), possibilitando inserção dinâmica de elementos e manipulação eficiente dos dados. Também foram implementadas funções de validação de datas, comparação entre registros, identificação de picos de casos e análise de variações entre diferentes períodos.

O projeto foi desenvolvido como atividade da disciplina de Estrutura de Dados I da Universidade Federal do Vale do São Francisco (UNIVASF).
